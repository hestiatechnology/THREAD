## THREAD and TextileEco

THREAD is an open standard — the schema, API contracts, validation rules, and CSV templates are free to use, implement, and build on.

[TextileEco](https://textileeco.com) is the company that publishes and maintains THREAD, and operates the hosted platform that implements it. Using TextileEco is the fastest way to get a THREAD-compliant system running, but any platform can implement the standard independently.

This separation is intentional. If other platforms adopt THREAD, the ecosystem grows. That's a feature, not a threat.
